# NiuBot

Persistent AI bot runtime — plug in any agent, get infinite conversation with long-term memory.

- infinite conversation flow with autonomous context management
- long-term memory: user preferences, complete session archives, message history
- built-in backends: Claude Code, Codex, Trae CLI, OpenCode, Cursor Agent, Pi
- built-in tools: tasks, contacts, messages, cron jobs
- configurable persona, multi-bot support
